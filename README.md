# Sasha Deejay

Official promo site for **Sasha Deejay** — Romanian DJ, producer, and former b-boy. Built to showcase music, biography, and latest YouTube uploads in one sleek, dark-themed experience.

## About the Artist

From Moldova to Reșița to Constanța — and from break-dance battles to club decks. Sasha Deejay has been shaping the Romanian hip-hop and electronic scene since the late '90s. Founder of the legendary West2Peu / K'Virus crew, turned label collaborator, turned international DJ. Four decades of rhythm, now behind the decks since 2005.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Cloudflare Pages (deploy via Wrangler)

## Features

- Responsive dark-themed landing page
- Biography timeline
- Latest album showcase
- YouTube recent uploads grid (auto-fetched via CORS proxy + local caching)
- Social platform links
